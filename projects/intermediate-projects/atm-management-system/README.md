# 🏧 ATM Management System in C

A console-based **ATM Management System** built using the **C programming language**.

This project simulates ATM operations such as deposit, withdrawal, balance inquiry, and PIN change using functions, conditions, loops, and global variables.

---

## ✨ Features

* 🔐 Login with PIN
* 💵 Deposit Money
* 💸 Withdraw Money
* 💰 Check Balance
* 🔑 Change ATM PIN
* 📞 Help / Enquiry Option
* 🚪 Exit System

---

## 🛠️ Technologies Used

* C Language
* Standard Input / Output Library (`stdio.h`)
