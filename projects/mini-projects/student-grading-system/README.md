# 🎓 Student Grade Calculator in C

A simple **Student Grade Calculator** built using the **C programming language**.

This beginner-friendly project calculates the average marks of a student based on multiple subjects and assigns a grade using conditional logic.

---

## ✨ Features

* 📚 Enter Number of Subjects
* 📝 Input Marks for Each Subject
* 📊 Calculate Average Marks
* 🏅 Automatic Grade Assignment
* 🔁 Repeat Calculation Option
* 🖥️ Console Based Program

---

## 🛠️ Technologies Used

* C Language
* Standard Input / Output Library (`stdio.h`)

---

## 📌 Grade Criteria

| Average Marks | Grade |
| ------------- | ----- |
| 90+           | A     |
| 80 - 89       | B     |
| 70 - 79       | C     |
| 60 - 69       | D     |
| Below 60      | E     |
