# 🧮 Calculator in C

A menu-driven **Calculator Program** built using the **C programming language**.

This beginner-friendly project performs basic arithmetic operations and power calculations using functions, loops, and conditional statements.

---

## ✨ Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* ⬜ Square of a Number
* 🧊 Cube of a Number
* 🚪 Exit Option
* ⚠️ Divide by Zero Handling

---

## 🛠️ Technologies Used

* C Language
* Standard Input / Output Library (`stdio.h`)
